# Fedora Windows VFIO Stability

This repository documents a Fedora host and Windows virtual machine configuration
using KVM/libvirt with full GPU passthrough.

The design targets near-native hardware performance while prioritizing stability,
predictability, and recoverability.

## Documentation

Primary design guides are included within the repository.
Supporting material includes appendices, scripts, Windows hardening resources,
and installation automation.

## Feedback

Technical review is welcome, particularly around stability, recovery scenarios,
and long-term maintainability.
